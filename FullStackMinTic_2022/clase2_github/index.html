1. git init para inicializar un repositorio
2. git add . para agregar todos los archivos a la lista de cambios
3. git commit -m "mensaje" para agregar un comentario a los cambios
4. git push -u origin master para subir los cambios al repositorio remoto
5. git pull origin master para descargar los cambios del repositorio remoto
6. git status para ver el estado de los archivos
7. git log para ver los cambios realizados
8. git diff para ver los cambios realizados
9. git diff --staged para ver los cambios realizados en el repositorio local
10. git diff HEAD~1 HEAD~2 para ver los cambios realizados entre dos commits
11. git diff HEAD~1 HEAD~2 --staged para ver los cambios realizados entre dos commits en el repositorio local
12. git diff HEAD~1 HEAD~2 --staged --word-diff para ver los cambios realizados entre dos commits en el repositorio local con diferencias de palabras
13. git diff HEAD~1 HEAD~2 --staged --word-diff --color-words para ver los cambios realizados entre dos commits en el repositorio local con diferencias de palabras y colores
14. git diff HEAD~1 HEAD~2 --staged --word-diff --color-words --color-diff para ver los cambios realizados entre dos commits en el repositorio local con diferencias de palabras y colores y diferencias de color
15. git diff HEAD~1 HEAD~2 --staged --word-diff --color-words --color-diff --color-words-space para ver los cambios realizados entre dos commits en el repositorio local con diferencias de palabras y colores y diferencias de color y espacios
16. git diff HEAD~1 HEAD~2 --staged --word-diff --color-words --color-diff --color-words-space --color-words-space-diff para ver los cambios realizados entre dos commits en el repositorio local con diferencias de palabras y colores y diferencias de color y espacios y diferencias de color de espacios
17. git branch para ver las ramas existentes
18. git branch nombre para crear una rama
19. git branch -d nombre para eliminar una rama
20. git branch -D nombre para eliminar una rama permanentemente
21. git remote add origin para agregar un repositorio remoto
22. git remote -v para ver los repositorios remotos
23. git remote -v -a para ver los repositorios remotos y locales
24. git remote -v -a -p para ver los repositorios remotos y locales con el protocolo de comunicacion
